# Digital Agency Website

A full multi-page digital agency website built in React, converted from a Figma design.

## Pages
- **Home** — Hero, services, stats, client logos
- **About Us** — Team, core values, pricing tiers
- **Services** — 6 services with deliverables, 4-step process
- **Portfolio** — Filterable project grid
- **Portfolio Details** — Case study with sidebar nav & results
- **Blog** — Featured post + article grid
- **Blog Details** — Full article with TOC, author sidebar, related posts
- **Contact** — Inquiry form with budget selector

## Stack
- React 18
- Vite
- Pure CSS-in-JS (no external UI library)
- Google Fonts (Inter + Inter Tight)

## Run locally
```bash
npm install
npm run dev
```
